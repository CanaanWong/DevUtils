## 目录结构

```
- dev.base          | 根目录
   - able           | 基类库接口相关
   - expand         | 基于 Base Activity、Fragment 扩展包
      - content     | Content Layout 基类
      - mvvm        | MVVM 架构基类
      - viewdata    | ViewDataBinding 基类
      - viewmodel   | ViewModel 基类
   - utils          | 依赖工具包
      - assist      | 功能辅助类 ( 抽取通用代码 )
```
